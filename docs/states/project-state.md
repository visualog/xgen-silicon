# Project State

Last updated: 2026-06-18

## Project

BrandGen / xGen is a local creative-tool project with a Next.js app, Electron
packaging path, shadcn/ui-based design-system route, and extensive task notes.

## Operating System

The repo now uses a lightweight AgentOS-compatible operating layer:

- `AGENTS.md` is the first-read router.
- `docs/states/` keeps current state and task board summaries.
- `docs/workflows/` keeps compact task-type workflows.
- `notes/` keeps detailed plans, reports, screenshots, and handoffs.

## Existing Local Priorities

- Respect the Next.js version warning before editing Next.js code.
- Keep Fabric-style notes and screenshots for meaningful implementation work.
- Use shadcn/ui as the active UI foundation.
- Use `docs/ui-style-system.md` before UI density, spacing, radius, and tone decisions.

## Non-Goals

- Do not unpack or overwrite the full AgentOS Enterprise/Lite/UltraLite package into the repo.
- Do not replace existing BrandGen notes or handoff practices.
- Do not introduce new package dependencies as part of operating-rule work.
